Homelab Infrastructure

Overview

This project documents the infrastructure of my homelab, including hardware components, network configuration, and software stack.

Hardware Components

  • Server: Dell PowerEdge R640
  • Storage: Synology NAS DS920+
  • Network: Ubiquiti UniFi Dream Machine Pro
  • Backup: Western Digital My Book 8TB

Network Configuration

The network is configured with VLANs for segmentation, a firewall for security, and a VPN for remote access.

Software Stack

  • Virtualization: Proxmox VE
  • Containerization: Docker
  • Monitoring: Prometheus and Grafana
  • Backup: Duplicati
